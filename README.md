
# Online Quiz System - Using PHP

The Online Quiz System using PHP is a web-based application designed to facilitate automated quiz-taking, allowing users to participate in quizzes, receive instant feedback, and view their scores, all managed through a user-friendly interface. The system includes features for quiz creation, result tracking, and user management.

## Features
-Quiz Creation: Admins can easily create and manage quizzes with different types of questions.

-Instant Results: Users get their scores immediately after completing the quiz.

-User Login: The system has secure login for admins or teachers, and students.
